像使用VIM一样使用Android Studio

Sync Project with Gradle Files

就我自己而言，感觉Android开发中修改构建脚本是一个相当高频的操作。 一旦修改，Android Studio马上会提示你需要同步：

不过貌似Android Studio并没有为”Sync now”配置快捷，所以我不得不拿鼠标点一下那个黄色提示条。这怎么行，赶紧给”Sync now”配置一个快捷键吧

配置方法：Preferences > Keymap > Plug-ins > Android Support > Sync Project with Gradle Files > Add Keyboard Shortcut

我设置的快捷键是”Shift+S”。从此以后”Shift+S”一下就可以轻松同步啦。
